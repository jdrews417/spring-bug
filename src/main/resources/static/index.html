Spring Security OAuth2 Bug Demo

This is a demonstration project to show a bug in the OAuth2 handling of spring security.

To see the bug in action, Click on the link here to visit the login page.

The page has 3 methods to login, form based login, and links to two OAuth sources.

No matter which login method you choose, you should end up at the same login form page (which will now indicate you are logged in).

If you end up at THIS page after authenticating, that is the undesired behaviour. The form login method works as expected, the OAuth2 methods do not.
